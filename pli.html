[PLI]

Bloco a ser executado antes da tag [SQL] (e após a tag [PAR], caso a mesma exista).

Este bloco pode possuir diversos comandos @> que podem ser executados e validados por #> <#

Quando não definido, o comando @>SQL é assumido como default.

Este bloco pode possuir diversos comandos @> que podem ser executados e validados por #> <#. Quando não definido, o comando @>SQL é assumido como default.

Exemplo de código

#> ‘[$OPCAO$]’ = ‘$%OPCAO%$’<#
[OPC]
DEL: %SVGI/md-close-circle-outline : Excluir Item : N

[PLE]
*ID#0

[PLF]
@>MKQ
+exemplosEOT[$OPCAO$]EOT[OPC]EOT[$ID$]EOT[PLE]

[PLI]
#> ‘[$OPCAO$]’ = ‘DEL’ <#
begin
delete titens where id = [$ID$];
end;

[SQL]
select id “ID#0”, desc_tecnica “Descrição#90” from titens
