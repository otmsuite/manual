[TIT]

Define o título a ser apresentado na tela que receberá parâmetros.

Normalmente esta tag não recebe validações, uma vez que quando a mesma é apresentada os parâmetros ainda não foram respondidos.

Se o texto finalizar com o caracter #, ao fechar a janela no botão vermelho no topo esquerdo da tela, uma mensagem de confirmação será solicitada.

Exemplo de código

[TIT]
Título da janela#

[SQL]
select id “$ID#10”, ID “Código#2”, Texto “Descrição#9$1”, Numero “Valor#N8$4@S”
from tmp_teste_edt X
order by id
