[MFF]

Mensagem condicional a ser executada após da execução de [PLF].
Caso seja necessário que a mensagem seja exibida somente após o clique em algum botão do grid, a tag [PLE] se faz necessária.

Exemplo de código

[OPC]
OPC01: %SVGF/solid/check : Confirmar : S
[MFF]
#> '[OPC]' = 'OPC01' <#
Ação executada com sucesso!

[PLE]
ID#0

[PLF]
#> '[OPC]' = 'OPC01' <#

begin
update titens set desc_tecnica = 'atualizou' where id = [PLE];
commit;
end;

[SQL]
select id "ID#0", desc_tecnica "Descrição#80" from titens
