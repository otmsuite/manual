Uso de UNION_TABLE

Permite que mais de um grid de dados, não necessariamente com o mesmo número ou tipo de colunas, seja exibido no relatório.

Deve ser utilizado entre os blocos de select.

OBS: Caso seja utilizado o UNION_TABLE, no browser e planilha serão considerados o select da tag [SQL].
