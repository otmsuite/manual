Seleção

É possível alterar o modo de seleção de registros. Para isso, usa-se uma coluna com o nome (!) seguido de um identificador

Sintaxe

ID "(!)Dados#5"

Os registros serão gravados na OTM_MENSAGEM com a sessão atual, TIPO igual ao nome da coluna e MSG com o conteúdo da coluna. Caso utilizado em conjunto de um [PLE] com seleção múltipla, serão habilitados dois botões no GRID (para selecionar e deselecionar os registros)

Exemplo de código

[DIM]
50:0:0:25:70:15:100
[SQL]
select id “ID#10”, ID “(!)Código#2”, Texto “Descrição#9$1”, Numero “Valor#N8$4@S”
from tmp_teste_edt X
order by id

Labels com caracteres especiais (como @ ou $) devem estar entre aspas duplas.

Exemplo de código

[SQL]
select id “ID#10”, ID “Código#5”, Texto “Descrição#15$1”
