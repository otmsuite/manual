DT

O parâmetro de data pode receber em sua notação:

N (opcional): Indica que o calendário não será exibido ao clicar no botão do mesmo (não é necessário ter ";" após o mesmo);

RangeInicial e RangeFinal (opcionais): somente podem ser utilizados juntos e indicam um intervalo de datas permitidas pelo parâmetro;

Where (opcional): somente serão consideradas como datas válidas as que retornarem verdadeiro na condição definida. Usa-se "[$EDT$]" como sendo a data a ser testada.

Sintaxe

[PARAMETRO]: Label: DT: N; Data Inicial; Data Final ;WHERE : Default : ID do VDD : [PAR Posicionamento]

Exemplo de código

[PAR]
[DATA]: Data : DT250 :NH-5 ;H+5 ; to _char(to_date(‘[$EDT$]’ ‘d’) in (2,3,4,5,6) : +1
