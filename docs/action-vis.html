[VIS]

Oculta/exibe um elemento (ação 0 oculta / ação 1 exibe).

Sintaxe

[TMP] 000 ; [VIS] ; elemento; ação

Exemplo

[MKP] Primeiro Título; 0; 0; 1430; 610; BCO; 1600x900; 0; 0;
[IMG] IMG_001; 610; 420; 170; 170; 000-000-000; [Brasil]
[TMP] 000 ; [VIS] ; IMG_001 ; 0
[TMP] 001; [VIS] ; IMG_001 ; 1
