@>GRF

Utilizado para gerar uma imagem de gráfico no formato .png.

Deve conter as seguintes informações:

Caminho do arquivo a ser gerado;

Os dados do gráfico;

Tipo do Gráfico (BAR, LIN, PIE, ARE, TER, GAU/DAS)

Cor de fundo do gráfico (R-G-B)

Dimensões da Imagem (largura , altura)

Para mais detalhes referentes à notação específica de cada tipo, verifique a documentação de MKPs.

Detalhe

É possível passar parâmetros dentro de { }, separados por ; no início do script.

Exemplo de código

[PLF]
@>GRF
%TEMP%/Grafico
Valor 1 = 100, Valor 2 = 200
PIE
TITULO
245-239-246
500, 500
