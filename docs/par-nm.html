Manual de Referência

NM

Parâmetro tipo numérico.

Recebe como sua notação:

  • Número de casas decimais;
  • Limite inferior/mínimo (mostrado em tooltiptext);
  • Pasta inicial para busca, arquivo default (conteúdo padrão para o campo) Limite superior/máximo (mostrado em tooltiptext).

Caso a extensão definida para os "arquivos" seja DIR, este parâmetro receberá pastas ao invés de arquivos.

Sintaxe

[PARAMETRO]: Label: NM: Número de Casas Decimais; Valor Mínimo;Valor Máximo : Default : ID do VDD : [PAR Posicionamento]

Detalhe

Opcionalmente pode receber o valor default (vide exemplo onde o valor default é 500).
OBS: Caso não seja implementado um VDD para o parâmetro, a Suite se encarregará de implementar automaticamente um VDD para validação do valor mínimo e máximo.

Se o indicador de casa decimais contiver B, uma barra de controle deslizante será apresentada.

Se o indicador B estiver acompanhado de R, o refresh será dinâmico.

Exemplo de código

[PAR]
[UPLOAD]: Carregamento de Arquivo: FL500: PDF; Upload; /Desktop : Arquivo_Upload.PDF
