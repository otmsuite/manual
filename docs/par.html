[PAR]

Esta tag, juntamente com a tag [SQL] é uma das principais funcionalidades do MKQ. Ela permite a criação de uma interface de entrada de dados para o usuário (tela de parâmetros).

Os comandos definidos em Tipos de Parâmetros (FL, CK, FK, DT, TX, NM, DC, PW) são utilizados nesta tag.

Para isso, utilizamos +> para iniciar cada definição de botão do [OPP]. OBS: Não há a necessidade de utilizar +> se todas definições do [OPP] forem de uma única linha cada.

Exemplo de código

[OPP]
#> instr(‘[$$$]’, “$$$”) = 0 and ‘[$$$]’ <> ‘0’ <# >
+>HIDDEN: %SVGF/solid/list-ol : Listar todos os itens : Sair
