CK

Parâmetro tipo check-box.Recebe na sua notação o valor para true (check marcado) e para false separados por ponto e vírgula.

Opcionalmente, dois valores textuais equivalentes a true/false podem ser especificados (para serem utilizados quando necessários).

O valor default deve ser idêntico a um dos dois valores definidos para true ou false.

Sintaxe

[PARAMETRO]: Label: CK: IDTRUE;IDFALSE;Texto TRUE;Texto FALSE : Default : ID do VDD : [PAR Posicionamento]

Exemplo de código

[PAR]
[CHECK]: Listar tabelas sem registros? : CK300 : S ; N ; Sim ; Não : S
