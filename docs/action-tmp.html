[TMP]

Executa um delay (em milissegundos) antes de executar um novo ciclo.

Sintaxe

[TMP] 000 ; [TIT] ; delay

Exemplo

[MKP] Teste; 0; 0; 1430; 610; BCO; 1600x900; 0; 0;
[TMP] 000 ; [SQL] ; [update otm_mensagem set msg=decode(msg,5,0,msg)+1 where tipo='PB' and sessao=SYS_CONTEXT('USERENV','SESSIONID')]
[TMP] 000 ; [TMP] ; 60000
