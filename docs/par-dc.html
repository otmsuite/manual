DC

Parâmetro tipo lista de opções. Recebe uma sequência de informações separadas por ponto e vírgula (ou ^).

Para cada par de dados, o primeiro será utilizado nos scripts enquanto que o segundo será apresentado na tela e no cabeçalho (quando utilizado).

A primeira opção é assumida como default (se nenhuma for passada).

Sintaxe

[PARAMETRO]: Label: DC: ID1;Texto 1;ID2;Texto 2;ID3;Texto 3 : Default : ID do VDD : [PAR Posicionamento]

Exemplo de código

[PAR]
[LISTAR]: Listar usuários : DC300 :C^Cadastrados^N^Não Cadastrados^T^Todos: Todos
