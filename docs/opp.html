[OPP]

Permite acrescentar botões de ação na tela de parâmetros (que normalmente possui apenas os botões Confirmar e Retornar). Deve ser utilizada em conjunto com a tag [PAR].

Caso use Limpar, ao retornar para a tela de parâmetros os parâmetros voltarão aos seus valores defaults.

Por padrão, os parâmetros mantêm os valores alterados. Após o clique, o [OPP] é considerado como um parâmetro, comumente sendo utilizado nos blocos [MII], [MIF], [PLI] e [SQL].

Seu conteúdo principal (ID), obtido através de [OPP], é equivalente ao identificador do botão clicado.

Seu conteúdo secundário (Texto), obtido através de {[OPP]}, é equivalente ao label do botão clicado.

Sintaxe

IDENTIFICADOR:IMAGEM:LABEL: SAIR (FECHAR FORM):LIMPAR

Exemplo de código

[OPP]
#> instr(‘[$$$]’, “$$$”) = 0 and ‘[$$$]’ <> ‘0’ <#
OPP01: SVGI/md-close-circle-outline : Confirmar : Sair

[PAR]
[CHECK]: Listar tabelas sem registros? : CK300 : S ; N ; Sim ; Não : S
