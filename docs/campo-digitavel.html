Campo digitável

Para facilitar nas buscas é possível tornar o campo da FK digitável. Dessa forma conseguimos filtrar os valores que aparecerão na LOV (lista de valores) ou mesmo selecionar algum registro sem ter que abrir a LOV.

Antes dos campos da LOV é possível definir um DIM para a FK. Os valores devem ser separados por espaço.

Sintaxe

Na sintaxe da FK, após o WHERE insira: 8=8^CAMPO_DE_BUSCA

Caso Tabela comece com *, habilita seleção múltipla.

Caso Tabela comece com !, altera o modo de seleção.

Exemplo de código

[PAR]
[ITENS]: Itens : FK350 : (select id, desc_tecnica from titens) ;
id “Código#30”, desc_tecnica “Descrição#70” ;
id||chr(9)||id||‘ - ’||desc_tecnica;WHERE and 8=8^desc_tecnica
