[ACT]

Executa atualização nos elementos MKQ, TXT, GRF, PIC, PDF, VID, AUD.

Sintaxe

[TMP] 000 ; [ACT] ; elemento ; [conteúdo para o elemento]; cor_de_fundo; cor_fonte

Os parâmetros COR_DE_FUNDO e COR_DA_FONTE são opcionais.

Quando informados, caso o elemento possua configurações de cores de fundo e/ou fonte, as mesmas serão alteradas.

Exemplo

Neste exemplo, a cada ciclo de tempo duas linhas de ações são executadas.

[MKP] Teste; 0; 0; 1430; 610; BCO; 1600x900; 0; 0;
[MKQ] MKQ_001; 20; 20; 760; 400; 250-000-000; [Modelos/MODELO_M]
[PIC] PIC_001 ; 800; 20; 200; 200; 250-200-000; [%SVGD/otmsuite|@ ^ N ^ Atualiza gráficos]
[TXT] TXT_001 ;1000; 20; 180; 100; 100-000-200; [OtmSuite ^ 250-250-250 ^ C/C/Noto Sans/35 ^ 100-100-000 ^ TootipText ^ X]
[TMP] 000 ; [ACT] ; TXT_001 ; [TESTE DE TEXTO^ 255-255-255 ^ C/C/Noto Sans/32 ^ 000-000-000]
[TMP] 000 ; [ACT] ; MKQ_001 ; [!TESTE]
[TMP] 000 ; [ACT] ; PIC_001 ; [%SVGD/otmsuite ^ RESIZE]
