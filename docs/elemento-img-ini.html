IMG_INI

Permite definir cor inicial para preenchimento de cada um dos segmentos de uma imagem indexada.

Sintaxe

[IMG_INI] Elemento; Segmento; ; [R-G-B]

Exemplo

[MKP] Teste ; 0; 0; 1430; 610; BCO; 1600x900; 0; 0;

[IMG] IMG_001 ; 200; 50; 500; 500; 000-000-000; [Brasil]

[IMG_INI] IMG_001; ; ; [AMR]

[IMG_INI] IMG_001; RS ; ; [VRM]
