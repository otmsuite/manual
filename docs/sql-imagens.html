UNION_TABLE_n

Uma imagem pode ser apresentada nas células do grid usando a sintaxe:

Sintaxe

#IMG^Nome_da_imagem^W^H

W e H, quando não informados, assumem default de 19x19.

Ao informar W e H poderá ser necessário aumentar a altura da linha (tag [DIM]).

Não é possível uso conjunto com colunas fixas ([DIM]).

Exemplo de código

[SQL]
select id “ID#10”, ‘{{X-X-X/X-X-X/Tahoma-0-14/H35}}’||descricao “Documento#90”,
decode(pf_pj,0, ‘FÍSICA’, ‘JURÍDICA’) “Pessoa#40”,
‘#IMG^%SVGD/X|’||decode(doc_obrigatorio, 0, ‘VRM’, ‘VDE’)||‘^22^22’ “Obrigatório#15”
from hyv_tipo_doc_tb
