FK

Parâmetro tipo lista de valores de uma tabela (grid).

A sequência de sua notação, contudo, não é tão complexa como parece à primeira vista.

RangeInicial e RangeFinal (opcionais): somente podem ser utilizados juntos e indicam um intervalo de datas permitidas pelo parâmetro;

Where (opcional): somente serão consideradas como datas válidas as que retornarem verdadeiro na condição definida. Usa-se "[$EDT$]" como sendo a data a ser testada.

Sintaxe

TABELA; CAMPOS_LOV; PK||chr(9)||DESCR; WHERE+ORDER : DEFAULT

Exemplo de código

[PAR]
[ITENS]: Itens: FK350 : (select id, desc_tecnica from titens) :
id “Código#30”, desc_tecnica “Descrição#70”;
id||chr(9)||id||‘ - ’||desc_tecnica
