Opções para habilitar botões na barra de ferramentas do PDF:

  • S - Salvar como
  • P - Imprimir
  • F - Localizar
  • O - Painel de opções
  • N - Navegação de páginas
  • Z - Zoom
  • A - Ajuste de página
  • R - Rotação de página
  • T - Ferramentas
  • L - Label de numeração de página
  • V - Modo de visualização
