Snippets

Os snippets são uma maneira incrivelmente poderosa de gerar rapidamente a sintaxe de código previamente definida a partir de um atalho. Funciona como um auto-complete, digita-se o início da palavra ou até mesmo uma abreviação e ao pressionar Enter a estrutura de código poderá ser substituída pelo conteúdo que o usuário determinar.

Ao digitar o atalho de um snippet uma caixa de contéudo aparecerá abaixo, contendo a documentação do conteúdo em questão.

O exemplo de snippet abaixo se trata de um Parâmetro do tipo Decode (Lista de opções), seu atalho é pardc.

Para conferir os snippets disponíveis clique no ícone destacado na imagem abaixo na barra de tarefas do atom.

Obs: serão mostrados os snippets de acordo com a linguagem definida no arquivo (MKQ ou MKP), para alterar use o atalho ctrl+shift+L.
