Action TMP

O arquivo .ACT (que possui o mesmo nome do arquivo .MKP) é responsável pela atualização dos elementos em função do tempo.

A cada ciclo do tempo definido em .MKP, um bloco sequencial de comandos temporais é executado.

Detalhe

Lembre-se que qualquer uma das ações temporais podem ser executadas pelos motores posicional ou acional.

Sintaxe

[TMP] identificador sequencial; natureza da ação temporal; sintaxe da ação temporal

Exemplo

Neste exemplo, a cada ciclo de tempo duas linhas de ações são executadas.

[MKP] Teste; 0; 0; 1430; 610; BCO; 1600x900; 0; 0;
[TMP] 000 ; [TIT] ; [Invisível]
[TMP] 000 ; [VIS] ; PIC_1 ; 0
[TMP] 001 ; [TIT] ; [Visível]
[TMP] 001 ; [VIS] ; PIC_1 ; 1
