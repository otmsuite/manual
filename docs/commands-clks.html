[CLK1L], [CLK2L], [CLK1R] e [CLK2R]

Declaram a definição de ações associadas ao clique no botão do mouse, podendo ser esquerdo (L) ou direito (R). O número de cliques é definido pelos numerais 1 (simples) ou 2 (duplo).

Sintaxe

[CLK1L] elemento; elemento; elemento; ...

Detalhe

O clique sobre um elemento pode disparar dois sub-comandos:

CLK_MEN - abre um menu suspenso que permite acionar .MKQ;

CLK_SQL - permite executar um .MKQ ou executar algum dos comandos [TMP].

Exemplo

[MKP] Teste ; 0; 0; 1430; 610; BCO; 1600x900; 0; 0;
[PIC] PIC_001 ; 800; 20; 200; 200; 250-200-000; [%SVGD/otmsuite|@ ^ N ^ CLK1L]
[COMMANDS]
[CLK1L] PIC_001
[CLK_SQL]
@>MKQ/TESTEA
