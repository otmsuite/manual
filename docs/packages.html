Packages

Como uma plataforma open-source, o Atom oferece diversos pacotes desenvolvidos pela própria comunidade. A Equipe Interna de Tecnologia da Otimiza desenvolveu três pacotes personalizados para a construção de aplicações na OTMSuite: otmsuite-utils, language-mkq e language-mkp. Para desenvolver aplicações somente um pacote é de uso obrigatório, o otmsuite-utils , entretanto, recomenda-se que os outros dois pacotes sejam instalados para uma experiência mais completa e visualmente agradável.

Para instalar um pacote, na aba de configurações vá até a opção Install e procure pelo nome do pacote desejado.
