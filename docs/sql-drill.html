DRILL

Para chamar outro MKQ através de dois cliques em determinada linha do grid, usa-se uma coluna com o nome DRILL#0 contendo o MKQ a ser aberto.

Sintaxe

'MKQ:MKQ_a_ser_chamado' DRILL#0

Antes de MKQ: é possível utilizar prefixos conforme o quadro do comando @>MKQ.

Após o nome do MKQ é possível inserir parâmetros separados por EOT, seguindo o padrão EOT chave EOT valor.
