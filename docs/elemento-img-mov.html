[IMG_MOV]

Define ação associada ao movimento do mouse sobre o segmento da imagem indexada.

Pintar segmento com a cor/efeito definido

Sintaxe da notação

[IMG_MOV] elemento; segmento; ; [R-G-B - Efeito]

Alterar outro(s) elemento(s) do painel

Sintaxe

[IMG_MOV] elemento; segmento; elemento a alterar; [conteúdo do elemento a alterar]

Executar .MKQ

Sintaxe

[IMG_MOV] elemento; segmento; elemento de referência; [@>MKQ arquivo .MKQ]

Exemplo

O exemplo abaixo utiliza a primeira e a segunda sintaxe.

[MKP] Teste ; 0; 0; 1430; 610; BCO; 1600x900; 0; 0;
[PIC] PIC_001 ; 800; 50; 200; 200; 229-245-251; [%SVGD/otmsuite|@ ^ N ^ Atualiza gráficos]
---- Define IMAGEM indexada
[IMG] IMG_001 ; 200; 50; 500 ; 500; 000-000-000; [Brasil]
--Uso da primeira sintaxe
[IMG_MOV] IMG_001 ; ; ; [000-000-000-4] --Uso da segunda sintaxe
[IMG_MOV] IMG_001 ; ; PIC_001 ;[#select case when 'X[ORI]'='X' then '%SVGD/otmsuite|@ ^ N ^ Atualiza gráficos' else '%SVGL/'||substr('[ORI]',1,1) end from dual]
