Editores de texto

Para o desenvolvimento de aplicações na OTMSuite é necessário o uso de um editor de texto. Atualmente a plataforma possui dois editores de texto homologados pela Equipe de Tecnologia: Atom e NotePad++. Recomenda-se que a escolha do usuário fique entre estes editores para uma experiência completa. Com o objetivo de facilitar o entendimento comum, a Linguagem MKQ, presente nestes editores validados, conta com elementos visuais que destacam seus blocos, comandos e caracteres.

Um recurso para facilitar o acesso ao código fonte de determinada aplicação (MKQ, MKP ou ETQ), até mesmo de um menu (MYO), é o atalho de teclado e mouse Ctrl+Alt+Botão Direito do mouse . Esta ação independe de qual editor de texto seja o escolhido pelo usuário, sempre abrirá o então definido como padrão da máquina.
