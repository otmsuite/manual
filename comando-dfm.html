Manual de Referência

@>DFM

Permite copiar, mover ou apagar arquivos de/para pastas do SO ou listar o conteúdo de uma pasta. Sua notação recebe:

Origem

Caminho e nome do arquivo de origem (a sofrer a ação).

Destino

Caminho e nome do arquivo/pasta destino (para ação Copiar e Mover).

Ação

indica a ação a ser realizada em origem (Copiar, Mover, Deletar ou Listar).

Interação/Hidden

Indica se haverá mensagem de interação com usuário caso arquivo destino já exista (para ação Listar, indica se lista arquivos/pastas ocultas/sistema).

Subpastas

Indica se deve criar pastas destinos caso não existam (para ação Listar indica se deve listar conteúdo das subpastas).

Erros_na_Tela

Indica se mensagens de erro devem ser apresentadas na tela (quando SIM, processo é interrompido). Ação Listar não considera esta notação

Mensagens de sucesso ou erro são armazenadas em OTM_MENSAGEM com tipo = 'DFM'. Na Ação Listar, a relação de arquivos/pastas será armazenada em OTM_MENSAGEM. Caso seja necessário utilizar : no caminho de origem ou destino, deve-se usar @.

Exemplo de código

[PLI]
@>SQL
begin
delete OTM_MENSAGEM where sessao=SYS_CONTEXT(‘USERENV’, ‘SESSIONID’) and tipo = ‘DFM’;
end;
@>DFM
C@/temp/teste1.txt : C@/temp/xxx/teste2.txt : C : S : S : N
@>DFM
C@/temp : : L : N : S
