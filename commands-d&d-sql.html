[D&D_SQL]

Executa ações equivalentes a CLK_SQL (permite executar um .MKQ ou executar algum dos comandos [TMP])

Sintaxe

[D&D_SQL] script

Exemplo

[MKP] Teste ; 0; 0; 1430; 610; BCO; 1600x900; 0; 0;
[TXT] TXT_001 ;1000; 20; 180; 100; 100-000-200; [OtmSuite ^ 250-250-250 ^ C/C/Tahoma/35 ^ 100-100-000 ^ TootipText ^ X]
[IMG] IMG_001 ; 610; 420; 170; 170; 000-000-000; [Brasil]
[COMMANDS]
[D&D] TXT_001 > IMG_001;GRF_006
[D&D_COR] 255-000-000 ; 000-250-000
[D&D_SQL] @>MKQ Modelos/Modelo_M
