Manual de Referência

[GRD]

Esta tag permite definir agrupamentos no cabeçalho, além de permitir redefinição de algumas configurações do grid.

Subtags

[OC]

Pode receber:

  • C

    Oculta o cabeçalho.

  • R

    Oculta o rodapé.

  • O

    Oculta os botões de ordenação.

  • F

    Oculta os botões de filtrar.

[FL]

A subtag [FL] permite realizar buscas em colunas ocultas caso seu conteúdo inicie com S (o padrão de buscas é somente em colunas visíveis).

[FT] e [FC]

Estas subtags permitem definir as fontes a serem utilizadas para o cabeçalho do grid de dados (tag [SQL]).

Notação das subtags de fontes:

Nome da fonte - Tipo - Tamanho

O Tipo segue a definição:

0 = Normal | 2 = Negrito | 3 = Ambos

Padrão assumido:

Noto Sans - 0 - 12
