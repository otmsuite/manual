Totalizador

O caractere @ indica que haverá uma linha de sumarização no cabeçalho do grid.

Pode ser seguido de C (count), S (sum), M (max), N (min), A (avg).

S e A só devem ser usados em colunas numéricas.

Labels com caracteres especiais (como @ ou $) devem estar entre aspas duplas.

Exemplo de código

[SQL]
select id “ID#10”, ID “Código#5”, Texto “Descrição#15$1”
