Formatação de célula

Padrão adotado para background (RGB):

254-254-254 e 231-238-245

Padrão adotado para foreground (RGB):

254-254-254 e 231-238-245

Fonte padrão:

Noto sans - 12

Cores e fontes

Para alterar a cor/fonte de uma célula, basta preceder o conteúdo conforme a sintaxe:
{RB-GB-BB/RF-GF-BF/Fonte/Tipo/Tamanho}

RB-GB-BB é o RGB do background. Use X-X-X (ou X) para adotar o padrão;

RF-GF-BF é o RGB do foreground. Use X-X-X (ou X) para adotar o padrão;

A Fonte deve estar cadastrada no mapeamento de fontes (ou instalada na máquina).

Tipo refere-se à fonte: 0-normal, 1-negrito, 2-itálico, 3-ambos

Se a string de formatação estiver dentro de chaves duplas, a definição será aplicada para a linha inteira a partir do campo em questão.

O último parâmetro pode definir a altura da linha onde o mesmo se encontra. Para isto, basta especificar Hn onde n é a altura. Esta definição prevalece sobre a definida na tag [DIM].
