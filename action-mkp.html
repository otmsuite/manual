[MKP]

Reconstrói o elemento passado como parâmetro.

Se a cor passada for [INI], retorna para cor inicial.

Sintaxe

[TMP] 000 ; [MKP] ; elemento ; [PX^PY^DX^DY^Cor_Fundo^Fonte]

Exemplo

[MKP] Teste; 0; 0; 1430; 610; BCO; 1600x900; 0; 0;
[IMG] IMG_001; 610; 420; 170; 170; 000-000-000; [Brasil]
[TMP] 000 ; [MKP] ; TXT_001 ; [ ^ ^ ^ ^0-178-0 ^C/C/Noto Sans/12]
