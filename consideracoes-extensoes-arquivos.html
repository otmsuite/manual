Manual de Referência

Considerações

  • Desenvolvimento realizado em Java versão 8 com suite NetBeans;
  • Edição de documentos preferencialmente em Atom ou Notepad++, com definição de linguagem MKQ;
  • Documentos salvos em padrão UTF-8;
  • A primeira linha de cada arquivo normalmente é desconsiderada (utilizar a mesma para comentários, que se iniciam com sequência de caracteres --);

Extensões

  • MKQ - MakeQuery;
  • MKP - MakePanel;
  • ACT - Action (para atualizar MKP);
  • SQL - (SCR descontinuado) Script (para gerar ACT dinamicamente);
  • MYO - Menus;
  • ETQ - Etiquetas;

  • Backup.JAR
